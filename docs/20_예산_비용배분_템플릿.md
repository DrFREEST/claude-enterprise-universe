## 6) ROI 측정 기준

### 비용 효율 지표
| 지표 | 계산식 | 목표값 |
|------|--------|--------|
| 산출물당 비용 | 총 비용 / 완료된 티켓 수 | <$50/티켓 |
| 에이전트 가동률 | 실행 시간 / 전체 시간 | >60% |
| 모델 적정성 | 복잡도-모델 매칭률 | >90% |
| 재작업률 | 재시도 티켓 / 전체 티켓 | <5% |

### 가치 측정
- 처리 속도: 티켓 완료까지 시간 (중앙값)
- 품질: 검증 통과율, 사후 결함률
- 고객 만족도: NPS, CSAT 점수
- 비용 절감: 기존 방식 대비 절감액

## 7) 예산 조정 프로세스

### 증액 요청
- 요청 주체: BU 리더
- 근거 자료: 처리량 증가 데이터, 긴급 프로젝트 계획
- 승인 경로: CFO 검토 → CEO 승인
- 승인 기간: 48시간 이내
- 증액 한도: 월간 예산의 20%까지

### 예산 이전
- BU 간 예산 이전 가능 (월 1회)
- 이전 한도: 각 BU 예산의 10%까지
- 승인: CFO 단독 승인 가능
- 조건: 이전 후에도 최소 예산(월 $3,000) 유지
