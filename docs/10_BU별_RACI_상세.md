# BU별 RACI 상세(업무 단위) — Enterprise 기준
- 작성일: 2026-02-12
- 표기: **R(수행)** / **A(최종승인)** / **C(협의)** / **I(공유)**
- 권한 원칙 요약:
  - **사업부(BU)**: L0~L1 (구현/도메인 QA)
  - **평가·품질본부**: L1(품질 판정/차단)
  - **운영본부(CAB/릴리즈/SRE)**: L2(배포/롤백)
  - **보안·정책·준법본부**: L3(정책/준법 판정)
  - **경영지원/인사/비서실**: 필요 시 L4 승인 경로

---

## 공통 참여 조직 축약
- **BU**: 해당 사업부 팀(기획/개발/도메인 QA/운영협업)
- **플랫폼**: 플랫폼본부(파이프라인/런타임/관측/비용)
- **품질**: 평가·품질본부(회귀/성능/게이트)
- **보안**: 보안·정책·준법본부(정책/보안기술/준법)
- **운영**: 운영본부(CAB/릴리즈/SRE)
- **DX**: 개발자경험본부(템플릿/SDK/문서)
- **CS**: 고객성공본부(온보딩/지원/공지)
- **비서/홍보**: 비서실/홍보·대외협력실
- **재무/계약**: 경영지원본부(재무/구매·계약)
